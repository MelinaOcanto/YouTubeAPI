Youtube API



Como consultar una api?

Para consultar una api debemos contar con la URL, el APIkey y el IDchannel. Si tenemos eso ya podemos iniciar el recorrido. En principio consultar una api implica que hay que realizar una solicitud "fetch" pero antes hay que declarar las variables contantes que utiliaremos.Una constante para la APIkey, otra para la URL y otra para el ID, como así también recoger el elemento html que mostrara los resultados.
Luego podemos realizar el "fetch" para manejar los datos de la respuesta. Alli utilizamos la palabra reservada .then que si la solicitud es exitosa convierte los datos a JSON. Elegimos las propiedades que queremos tomar y realizamos un forEach que itera la lista de videos. Para cada video creamos un iframe .Para que el mismo sea mostrado debemos crearlo con createElemtent para agregarlo al DOM. Es por eso que que agregamos un appendChild al contenedor que aloja los videos.


Este es un pequeño resumen de como realizar una consulta a una api.
